# Aesthetic Coffee Landing Page

A stylish and modern coffee-themed landing page built with HTML and CSS. This project showcases a warm, elegant coffee experience through a hero section, story content, featured products, and a polished footer.

## Features

- Clean and attractive coffee-themed design
- Navigation bar with smooth section links
- Hero section with a bold call-to-action button
- "Our Story" and coffee experience sections
- Product cards highlighting popular coffee options
- Responsive layout for smaller screens

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Playfair Display)

## Project Structure

- index.html: Main structure of the landing page
- style.css: Styling, layout, colors, and responsive behavior

## How to Run

1. Open the project folder in your browser.
2. Launch index.html directly, or use a live server extension such as Live Server in VS Code.

## About

This project is a simple static website designed to create a premium coffee brand feel using only front-end web technologies.
